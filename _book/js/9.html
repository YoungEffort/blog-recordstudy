Error

Error

错误处理：是代码运行时的代码错误处理

  1. try···catch语句：捕捉运行代码出错，不会阻止后续代码的执行

     try{
     // 可能会导致错误的代码
 } catch(error){
     // 在错误发生时怎么处理
 }

  2. 自定义错误处理：throw new Error()---Error可替换成其他错误类型，提示开发者作用，执行此语句，阻断后续代码执行

  3. javascript代码执行分为预编译和代码执行阶段： 预编译阶段即浏览器语法检测解析阶段，如若遇到不支持的语法，一行js代码都不会执行，过了预编译阶段，代码才能执行
